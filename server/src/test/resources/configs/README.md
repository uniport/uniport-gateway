Examples for Portal-Gateway
===

All examples should (by do not yet) contain the same directory structure:

```
├── config
│   ├── dynamic-config/
│   |    ├── service1/service1.json
│   |    └── service2/service2.json
│   └── uniport-gateway.json
├── uniport-gateway.env
└── README.md
```
